# Simple Neural Network Without Machine Learning Libraries

## 📌 Overview
This project implements a **basic neural network from scratch** without using any Python machine learning libraries like TensorFlow, PyTorch, or Scikit-learn.

It consists of:
- **1 input layer** (2 neurons)
- **1 hidden layer** (2 neurons)
- **1 output layer** (1 neuron)

The network uses the **sigmoid activation function** and is trained using **manual backpropagation**.

## 🚀 Features
✅ **No external ML libraries** – only `math` and `random`  
✅ **Fully connected 2-layer neural network**  
✅ **Manual implementation of feedforward and backpropagation**  
✅ **Solves XOR problem using a neural network**  

## 🏗️ Tech Stack
- **Python** (Built-in `math` and `random` modules)
